## Linux Installation
1. `apt install git -y`
2. `apt install python3 -y`
3. `git clone https://github.com/Mrp1r4t3/FBDO`
4. `cd FBDO`
5. `pip3 install requests`
6. `pip3 install bs4`
7. `python3 fbd0.py`
